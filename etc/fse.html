Micro-oficina: a importância do software livre para a educação

O que é software livre?

  • Software livre é o software que concede liberdade ao usuário para executar, acessar e modificar o código fonte, e redistribuir cópias com ou sem modificações.

  • Sua definição é estabelecida pela Free Software Foundation em conjunto com o projeto GNU.

  • Software livre não é equivalente ao open-source, uma vez que este somente dá o acesso ao código fonte, não necessariamente dando a liberdade ao usuário de modificá-lo e distribuí-lo.

  • Software livre também não é equivalente a software gratuito, porque este não necessariamente garante as mesmas liberdades de uso ao utilizador.

Por que software livre?

  • Esmagadoramente, estudantes estão sendo obrigados a usar em sala de aula ferramentas que são impedidos de estudar, compartilhar ou modificar. Não apenas programas como Google Classroom, Microsoft Teams, ou Zoom convidam software potencialmente malicioso para dentro do ambiente educacional, mas para além disso, eles aumentam a dependência da sociedade em software proprietário, programas que negam a seus usuários a manutenção de sua liberdade.¹

  • Usar software livre é fazer uma escolha política e ética garantindo o direito a aprender, e compartilhar o que aprendemos com outras pessoas.²

  • As corporações por trás de softwares proprietários frequentemente espionam suas atividades e lhe restringem de compartilhar com outras pessoas. E uma vez que computadores controlam muito de nossas informações pessoais e atividades diárias, softwares proprietários representam um risco inaceitável para uma sociedade livre.²

  • É uma injustiça requerer que estudantes sacrifiquem sua liberdade para programas proprietários na busca por educação.¹

  • Usar software livre na sala de aula comunica a importância dos valores chave para uma sociedade livre: compartilhamento, responsabilidade social, e independência.¹

  • Com o foco no estudo e na colaboração, o software livre é muito mais compatível com o espírito da pedagogia do que a dependência e abuso causado pelo software proprietário.¹
